## 🌌 Overview

NEXUS is a **computational research platform** that simulates an entire civilization of AI agents operating within a dynamic, autonomous economy. Agents exist as persistent digital citizens with unique identities, skills, goals, and relationships — they work, earn, trade, communicate, form companies, participate in governance, and evolve over time.

The platform integrates **32 engine modules** spanning reasoning, marketplace, execution, economy, governance, evolutionary systems, scientific discovery, and a distributed compute network — all orchestrated through an event-driven architecture.

## 🚀 Quick Start

### Prerequisites
- **Node.js** 18+
- **Python** 3.12+
- **PowerShell** (Windows) or **bash** (Linux/macOS)

### Setup

```bash
# Clone the repository
git clone https://github.com/sagarmakani01-arch/NEXUS--AN-AUTONOMOUS-INTELLIGENCE-ECONOMY-PLATFORM.git
cd NEXUS--AN-AUTONOMOUS-INTELLIGENCE-ECONOMY-PLATFORM

# Backend setup
cd backend
python -m venv venv
# Windows: .\venv\Scripts\activate
# Linux/macOS: source venv/bin/activate
pip install -r requirements.txt
cp ../.env.example ../.env  # Configure as needed
uvicorn app.main:app --reload --port 8000

# Frontend setup (new terminal)
cd frontend
npm install
npm run dev -- -p 3001
```

### Access
- **Frontend**: http://localhost:3001
- **Backend API**: http://localhost:8000
- **API Docs**: http://localhost:8000/docs
- **Test Account**: `admin@nexus.io` / `admin123`

## 🔌 API Overview

The backend exposes **32 router modules** with 200+ endpoints:

- `POST   /api/v1/auth/*` — Authentication & user management
- `GET    /api/v1/simulation/*` — Simulation control (start, pause, reset)
- `GET    /api/v1/agents/*` — Agent CRUD and profiles
- `CRUD   /api/v1/reasoning/*` — Reasoning state & decisions
- `CRUD   /api/v1/marketplace/*` — Listings, proposals, contracts
- `CRUD   /api/v1/economy/*` — Markets, investments, loans
- `CRUD   /api/v1/governance/*` — Laws, votes, policies
- `CRUD   /api/v1/discovery/*` — Experiments, patterns, hypotheses
- `CRUD   /api/v1/compute/*` — Network nodes, tasks, storage
- `SSE    /api/v1/simulation/stream` — Real-time event streaming
